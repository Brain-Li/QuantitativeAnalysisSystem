# 字段展示规则说明

股票列表与趋势分析共用一套**展示层**规则，实现代码：`src/app/utils/fieldDisplayFormat.ts`。趋势分析**纵轴 domain** 见 **`computeTrendYAxisDomain`**、**`getZhangDieFuYDomainByStockCode`**；是否**含 0 轴 / y=0 参考线** 见 **`trendYAxisShouldIncludeZero`**。图表仍按**原始数值**绑轴。

---

## 一、股票列表模块

| 序号 | 字段 | 规则 |
|------|------|------|
| 1 | **日期** | 具体到**日**，去掉时分秒。 |
| 2 | **涨跌幅** | 原始值 **÷100** 后再以百分数展示，**两位小数**；+ 红 / − 绿。 |
| 3 | **换手率%** | 表头「换手率」；先 **×100** 再百分数展示，**两位小数**（实现见第五节）。 |
| 4 | **成交量** | **万**单位，**两位小数**。 |
| 5 | **波动率** | 百分数，**两位小数**（原始值 ×100 + `%`）。 |
| 6 | **BETA** | **两位小数**。 |
| 7 | **相关性** | **两位小数**。 |
| 8 | **macd_signal** | **三位小数**。 |
| 9 | **slowkdj_signal** | 列名含 **slowkdj_signal**（或归一化后含 `slowkdj`）：**三位小数**（与 macd_signal 展示一致）。 |
| 10 | **总分** | 列名含「**总分**」：**一位小数**。 |
| — | **其余字段** | **按原始值展示**。 |

---

## 二、趋势分析模块（指标数值展示）

与**第一节**一致；Tooltip / 纵轴刻度未分类数值为 **两位小数**（见原第三节说明）。**涨跌幅** Tooltip 红/绿色。

---

## 三、趋势分析：纵轴 domain 与 0 轴（核心）

以下对**当前所选股票代码**（主图、对比图各取该图已选第一只股票代码）生效。

### 1. 总分

- **展示**：**一位小数**（`zongfen`）。
- **纵轴**：数据 min/max 加 padding 后**强制包含 0**；`y=0` 参考线（区间含 0 时绘制）。

### 2. 涨跌幅

- **（1）** 代码以 **60**、**00** 开头（A 股主板）：纵轴 domain 固定 **`[-10, 10]`**（对应约 ±10% 刻度，与 Excel 存数一致）。
- **（2）** 代码以 **688**、**300** 开头（科创板、创业板）：纵轴 domain 固定 **`[-20, 20]`**（约 ±20%）。
- **（3）** 其余代码：默认 **`[-10, 10]`**。
- **0 轴**：domain 含 0；绘制 **`y=0`** 参考线。

> 匹配规则为字符串 **前缀**：先判断 `688`、`300`，再判断 `60`、`00`。

### 3. 开盘、最高、最低、收盘（及常见英文列名）

- **列名匹配**：字段名包含 **开盘 / 最高 / 最低 / 收盘** 任一；或归一化后（去空格、小写、去下划线）为 **`open` / `high` / `low` / `close`**。
- **纵轴**：仅按该列数据 min/max **加 padding**，**不**强制包含 0；**不**绘制 **`y=0`** 参考线（避免价格区间被压扁、走势不明显）。

### 4. 其余条文指标（含 0 轴，含波动率、`lst_close`）

对 **换手率、放量天数、平均量比_50天、成交量、放量天数_volume、平均量比_50天_volume、波动率、`lst_close`、BETA、相关性、总市值、长期、短期、macd_signal、slowkdj_signal** 及**未单独排除**的其它数值列：

- **纵轴**：padding 后**强制包含 0**（`lo = min(0, lo)`，`hi = max(0, hi)`）。
- **0 轴**：当 `下限 ≤ 0 ≤ 上限` 时绘制 **`y=0`** 虚线参考线。

**总分**见上文 §1。

### 实现与双轴

- 函数：`computeTrendYAxisDomain`；含 0 判断：`trendYAxisShouldIncludeZero(metric)`。
- 双轴时左右指标分别计算 domain；**涨跌幅** 均按该股票代码取 ±10 / ±20；**参考线**按左右指标各自是否「应含 0」决定是否绘制。

## 五、换手率实现说明（条文与代码对齐）

- 原始值 **≥ 0.01**：`(原始值 × 100).toFixed(2)%`。
- 原始值 **小于 0.01**：`新值 = 原始值 × 100`；若 **|新值| 小于 0.15** 再 ×100 → 如 `12.55%`。
- **|新值| ≥ 0.15**（如 `0.008`）：不再第二次乘 → `0.80%`。

函数：`formatTurnoverPercentDisplay`。
